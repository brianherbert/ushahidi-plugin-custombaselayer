== Description ==
Configure your own baselayer. This is useful for deployers running their own tileservers, people with Mapbox accounts with custom baselayers, etc.

== Installation ==
1. Copy the entire /custombaselayer/ directory into your /plugins/ directory.
2. Activate the plugin.
3. Input the proper settings on the addons settings page.
4. Select your new layer from the dropdown on the map settings page.
